Importing an existing project¶

Crown projects are just regular folders. If you want to import an existing project, click the Import... button in the projects list:

Then, select the folder containing the project and click the Open button:

If the project is correctly recognized, Crown will add it to the projects list:

When you are ready, click the Open button on the newly imported project to start working on it.
