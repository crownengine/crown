================
Installing Crown
================

The recommended way to install Crown is by downloading a binary package for your
operating system from the official website. Such package can be extracted
anywhere on your local machine and its contents used directly.

Installing different versions simultaneusly is also supported when using this
method.

Hassle-free
===========

Crown requires no setup or account, and does not need an internet connection
during use. We don't collect any data and there are no ads. You can just
download it, unzip it and start creating right away.

Software dependencies to run Crown are minimal and almost certainly you won't
need to install anything else to get started.
