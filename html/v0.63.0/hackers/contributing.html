Contributing¶

Everyone is welcome to contribute to Crown. You can help by submitting patches, testing features, promoting the project online, sponsoring it, and more.

Legal Requirements¶

Crown is distributed under different licenses for different parts of the project:

  • Runtime

  • Tools

  • Manual

By submitting a contribution, you state that you have the right to do so and that the work does not infringe any copyright or violate any license. If accepted, it will be distributed under the terms that apply to the part of Crown it modifies.

Submitting Patches¶

For trivial patches, just send a pull request on GitHub. For more complex changes, it is better to discuss the idea beforehand by joining our Discord server or opening an issue on GitHub.
